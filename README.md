# ffmpeg.aui2

libavformat、libavcodecなどを使用して、動画を読み込むAviUtl2の入力プラグインです。

> [!WARNING]
> [L-Smash Works](https://github.com/Mr-Ojii/L-SMASH-Works)と違い、このプラグインはユーザー数が少なく、バグが多い可能性があります。覚悟してください。

## インストール

[Releases](https://github.com/sevenc-nanashi/ffmpeg.aui2/releases/latest)から`sevenc-nanashi.ffmpeg-aui2-v{{version}}.au2pkg.zip`をダウンロードし、AviUtl2のプレビューにドラッグ&ドロップしてください。
